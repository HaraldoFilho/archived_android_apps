Bookmarks

App bar

Displays the page name and has three options:

  Go back to the main screen.
  Sort bookmarks.
  Open the options menu.

List

There are two operations that you can do on a list item:

Click
Set the location on the map. Opens an alert dialog if a location is already set in the map (button is GREEN).

Press for a few seconds
Opens a menu with four options:

Share
Opens a menu where you can choose between some applications to share the location through a url that can be opened on the Google Maps™ app.

Edit name
Opens a dialog window on which you can change the location name.

Edit address
Opens a dialog window on which you can change the location address.
Opens a dialog window on which you can delete the location from the list.

Options menu

 Sort
Alphabetically sort the bookmarks list.

Backup
Make backup of bookmarks.

Restore
Restore backup of bookmarks.

Help
Displays this help screen.
