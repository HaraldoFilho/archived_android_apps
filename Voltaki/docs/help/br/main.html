Mapa

Mostra a localização atual se o botão está AMARELO ou uma localização marcada se o botão está VERDE. O mapa possui um botão flutuante verde que se clicado faz com que o marcador vermelho retorne para o centro do mapa, ou se pressionado por alguns segundos quando o botão está AMARELO atualiza o mapa.

Botão

O botão tem seis estados com o seguinte código de cores:

VERMELHO (DESCONECTADO)
As configurações de localização estão desligadas, assim não há nenhum provedor de localização disponível e o aplicativo não pode obter a localização atual, mas você pode configurar locais no mapa através das listas de marcadores e histórico. Você deve ligar as configurações de localização e reiniciar o aplicativo para obter a localização atual. Você pode pressionar o botão por alguns segundos para fazer a reinicialização.

LARANJA (OBTENDO LOCALIZAÇÃO...)
O aplicativo está obtendo a localização atual e o botão está desabilitado para cliques. Quando a localização é encontrada o botão se torna AMARELO.

AMARELO (VOLTA AQUI)
Se você clicar o botão o local atual será marcado, o botão se tornará VERDE e um ícone  de notificação aparecerá na barra de estado. Clicando no ícone o aplicativo Google Maps™ será aberto e você poderá navegar para o local marcado. Se você pressionar o botão por alguns segundos enquanto ele estiver AMARELO uma caixa de diálogo se abrirá, onde você poderá adicionar o local atual à lista de marcadores. Você pode adicionar um nome para o local ou deixar em branco. No último caso, a data e horário atuais serão adicionadas como o nome do local.

VERDE (VOLTAR)
Se você clicar o botão o aplicativo Google Maps™ será aberto e você poderá navegar para o local marcado. Esse pode ser o último local marcado ou um local que você escolheu da lista de histórico ou dos marcadores. Se você pressionar o botão por alguns segundos, uma caixa de diálogo se abrirá, onde você poderá reiniciar o aplicativo para o estado inicial. Nesse caso, o botão se tornará LARANJA novamente e o aplicativo irá procurar a localização atual. Quando a localização for encontrada, o botão se tornará AMARELO e o mapa exibirá um marcador no local atual.

VERDE (VOLTAR)
O mesmo comportamento do estado anterior, a cor amarela das letras apenas indica que o botão já foi clicado e você provavelmente está voltando para o local marcado. Depois que você voltar para o seu local de origem você precisará reiniciar o aplicativo através do menu de contexto ou pressionando o botão por alguns segundos.

VERDE (VOLTAR)
O mesmo comportamento dos dois estados anteriores, a cor vermelha das letras apenas indica que as configurações de localização estão desligadas, assim não há nenhum provedor de localização disponível. Você pode clicar o botão para abrir o aplicativo Google Maps™ mas não será possível usar a navegação a menos que você ligue as configurações de localização.

Menu de opções

 Adicionar marcador
Abre uma caixa de diálogo onde você poderá adicionar o local no mapa na lista de marcadores. Você pode adicionar um nome para o local ou deixar em branco. No último caso, a data e horário atuais serão adicionadas como o nome do local.

 Compartilhar
Abre um menu em que você pode escolher dentre vários aplicativos para compartilhar o local no mapa através de um url que pode ser aberto no aplicativo Google Maps™.

Reiniciar (quando o botão está VERDE)
Abre uma caixa de diálogo onde você poderá reiniciar o aplicativo. Nesse caso, se as configurações de localização estiverem ligadas,  o botão se tornará LARANJA novamente e o aplicativo irá procurar a localização atual. Quando a localização for encontrada, o botão se tornará AMARELO e o mapa exibirá um marcador no local atual. Se as configurações de localização estiverem desligadas o botão se tornará VERMELHO.

Reiniciar (quando o botão está VERMELHO)
Mesmo comportamento anterior, mas a caixa de diálogo não se abrirá.

Atualizar (quando o botão está AMARELO)
Mesmo comportamento de Reiniciar, mas a caixa de diálogo não se abrirá.
