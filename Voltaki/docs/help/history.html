History

App bar

Displays the page name and has three options:

  Go back to the main screen.
  Clear history list.
  Open the options menu.

List

There are two operations that you can do on a list item:

Click
Sets the location on the map. Opens an alert dialog if a location is already set in the map (button is GREEN).

Press for a few seconds
Opens a menu with the following options:

Add to bookmarks
Opens a dialog window on which you can add the location to the bookmarks list. You can type a name to the location or leave the text field blank. In the latter case, the current date and time will be assigned as location name.

Share
Opens a menu where you can choose between some applications to share the location through a url that can be opened on the Google Maps™ app.

Delete
Opens a dialog window on which you can delete the location from the list.

Options menu

 Clear history
Opens a dialog window on which you can delete all items in the history.

Help
Displays this help screen.
