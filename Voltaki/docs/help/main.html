Opens the options menu.

Map

Displays the current location if the button is YELLOW or a marked location if it is GREEN. The map has a green floating button that if clicked bring back the red marker to the center of the map, or if pressed for a few seconds while the button is YELLOW refresh the map.

Button

The button has six states with the following color code:

RED (OFFLINE)
The location settings are turned off, so there is no location provider available and the app can not get the current location. But, you can set locations on the map through the history and bookmarks lists. You must turn on the location settings and reset the application in order to get the current location. You can press the button for a few seconds to do the reset.

ORANGE (GETTING LOCATION...)
The app is getting the current location and the button is disabled for clicks. When location is found the button turns YELLOW.

YELLOW (COME BACK HERE)
If you click the button the current location will be marked, the button turns GREEN and an notification icon will appear on the status bar. Clicking on the notification will open Google Maps™ app and you can navigate to the location marked. If you press for a few seconds the button when YELLOW, a dialog window will open on which you can add the current location to Bookmarks. You can type a name to the location or leave the text field blank. In the latter case, the current date and time will be assigned as location name.

GREEN (GO BACK)
If you click the button the Google Maps™ app will be opened and you can navigate to the location marked. This can be the last location marked or a location that you chose from History or Bookmarks. If you press for a few seconds the button, a dialog window will open on which you can reset the app to the initial state. In this case, the button turns ORANGE again and the app will search for the current location. When the location is found, the button turns YELLOW and the map will display a marker on the current location.

GREEN (GO BACK)
Same behavior of the previous state, the yellow color on the letters just indicates that the button was already clicked and you probably is going back to the location marked. After you are back to your origin location you need to reset the app through context menu or pressing the button for a few seconds.

GREEN (GO BACK)
Same behavior of the two previous states, the red color on the letters just indicates that the location settings are turned off, so there is no location provider available. You can click the button to open the Google Maps™ app but it will be not possible to navigate unless you turn the location settings on.

Options menu

 Add to bookmarks
Opens a dialog window on which you can add the location on the map to the bookmarks list. You can type a name to the location or leave the text field blank. In the latter case, the current date and time will be assigned as location name.

 Share
Opens a menu where you can choose between some applications to share the location on map through a url that can be opened on the Google Maps™ app.

Reset (when button is GREEN)
Same behavior, but the dialog window will not be opened.

Refresh (when button is YELLOW)
Same behavior of Reset, but the dialog window will not be opened.
