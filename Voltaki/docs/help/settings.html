Settings

App bar

Displays the page name and has three options:

  Go back to the main screen.
  Restore default settings.
  Open the options menu.

Settings options

Map type
Choose between three map types:
  • Normal: Typical road map. Roads, some man-made features, and important natural features such as rivers are shown. Road and feature labels are also visible.
  • Hybrid: Satellite photograph data with road maps added. Road and feature labels are also visible.
  • Satellite: Satellite photograph data. Road and feature labels are not visible.
Navigation option
Choose how you will go back to a location:
  • Map: Opens the marked location on Google Maps™ app. Selecting this option, you still can go to Google Maps™ Navigator clicking on the directions button. Tip: Clicking on the compass button will make easier to see on which direction the marked location is.
  • Navigator: Opens the marked location directly on Google Maps™ Navigator.

Default navigation mode
Choose on which mode the Google Maps™" or Google Maps™ Navigator apps will be set when opened. The options are: Walk, Drive and Bicycle. These modes can be changed later directly on the Google's apps.

Default zoom level
Choose the default zoom level on map when it is updated with a new location. The options are: Auto, High, Mid and Low. If Auto is chosen the zoom level will be defined according to the default navigation mode and the modes Walk, Bicycle and Car are respectively equivalent to levels High, Mid and Low.

Notification
Choose if a notification is shown when the YELLOW button is pressed. Clicking on the notification opens the marked location on Google Maps™ app.

Button vibrate feedback
Choose if the device will vibrate or not when the button is pressed.

Button long click
Enabling this option, on pressing the button for a few seconds you can add the current location to bookmarks if button is YELLOW or reset the app if it is GREEN.

Bookmarks backup
Choose if backup of bookmarks will be done automatically every time a new location is added to the list. Backup will not be done when a location is deleted from the list, but at every backup all previously deleted locations will be lost.

Maximum history items
Set the maximum number of items in the history list. The options are: 10, 100 or Unlimited. The greater the number of items, more memory will be needed to store the list. Older items that exceed the maximum limit will be automatically deleted.  

Options menu

 Restore default settings
Restore the settings to the default values.
