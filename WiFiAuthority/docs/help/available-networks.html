Available Networks

Application Bar

Has three options:

  Go back to the "My Networks" screen.
  Add a hidden network.
  Open the options menu.

"Available Networks" list

Lists all the available networs in the location with the following information:

SSID (turns green when connected)
Network security
Signal level

Has also the following indication icons:

Configuration
Network is already configured on device

Security
High security (WPA/WPA2/FT PSK criptography)
 Medium security (WEP criptography)
 Unsecure network (open network without criptography)

Signal  level
Strong signal
 Weak signal
 Very weak signal

There is only one operation you can execute on a list item:

Click
Opens a dialog box where you can type the network's password, insert a description (optional) and connect to the network.

Options menu

  Add hidden network
Opens a dialog box where you can connect to a hidden network. You must type the network name (SSID), choose the security type and, if you wish, insert a description.

Settings
Shows the application settings screen.

Help
Shows this help screen.
