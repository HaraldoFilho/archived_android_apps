Redes Disponíveis

Barra do aplicativo

Tem três opções:

  Volta para a tela "Minhas Redes".
  Adiciona uma  rede oculta.
  Abre o menu de opções.

Lista "Redes Disponíveis"

Lista todas as redes disponíveis no local com as seguintes informações:

SSID (fica verde quando conectada)
Segurança da rede
Nível do sinal

Tem ainda os seguintes ícones de indicação:

Configuração
Rede já está configurada no dispositivo

Segurança
Segurança alta (criptografia WPA/WPA2/FT PSK)
 Segurança média (criptografia WEP)
 Rede insegura (rede aberta sem criptografia)

Nível do sinal
Sinal forte
 Sinal fraco
 Sinal muito fraco

Há apenas uma operação que você pode executar em um item da lista:

Clique
Abre uma caixa de diálogo onde você pode digitar a senha da rede, inserir uma descrição (opcional) e conectar à rede.

Menu de opções

Adicionar rede oculta
Abre uma caixa de diálogo onde você pode se conectar a uma rede oculta. Você deve digitar o nome da rede (SSID), escolher o tipo de segurança, inserir a senha e, caso deseje, inserir uma descrição.
